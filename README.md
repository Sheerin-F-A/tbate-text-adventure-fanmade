# T.B.A.T.E. - Player One (Text Adventure)

A retro, 8-bit inspired choice-based text adventure game based on the early arcs of *The Beginning After The End*. 

Players can experience the journey of King Grey reincarnating as Arthur Leywin through an interactive, vaporwave-aesthetic UI. Built purely with HTML, CSS, and Vanilla JavaScript.
